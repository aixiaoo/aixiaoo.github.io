# NAT服务器使用方法

# 1. 什么是NAT服务器

网络地址转换（英语：Network Address Translation，缩写：NAT；又称网络掩蔽、IP掩蔽）在计算机网络中是一种在IP数据包通过路由器或防火墙时重写来源IP地址或目的IP地址的技术。

这种技术被普遍使用在有多台主机但只通过一个公有IP地址访问互联网的私有网络中。它是一个方便且得到了广泛应用的技术。当然，NAT也让主机之间的通信变得复杂，导致了通信效率的降低。

简单地说，NAT服务器就是没有独立IP的服务器。

# 优势：

价格便宜

除了共享IP之外，NAT服务器与其他服务器没有太大差别。
