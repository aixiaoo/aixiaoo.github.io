    # 重置流量

    # Q：流量用完了怎么办？

    # A：可以选择等待到下一个月（周期）或者是付费提前重置流量。如果离下一次重置（账单日）比较近，建议等待账单日重置。

    流量重置是指，通过额外地支付与套餐一个月的等值金额，直接重置本次月（周期）流量额度。

    举个例子，你的账单付款日在20号，今天10号，你的流量只剩500MB了。假如你是25元的套餐，你可以通过支付25元，来重置本个月（周期）的流量，重置完后，你的流量就会变成25G。这样你就可以在接下来的10天里继续使用。

    重置后的流量，不会与之前没用完的叠加，也不会留到下一个月，随着本个月周期的结束而无效。

    # Q：如何重置流量？

    # A：在用户中心-提交工单页面

    # 选择通用工单，优先级为中，服务选择对应需要重置流量的机器。
